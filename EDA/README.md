# EDA детекция аномалий в логах
### Основная часть:
1. Характеристика логов: количество записей, структура, поля.
2. Графики распределения логов во времени (по компонентам).
3. Частотный анализ сообщений (ключевые слова, биграммы).
4. Результаты template mining:
    1) количество уникальных шаблонов;
    2) распределение по частоте;
    3) выделение редких template_id.
5. Последовательности шаблонов и примеры подозрительных цепочек.
6. Кластеризация:
    1) TF-IDF + KMeans/DBSCAN по векторизованным сообщениям;
    2) PCA / t-SNE для визуализации кластеров логов.
  
### Дополнительные:
1. Временные паттерны (автокорреляция, всплески активности).
2. Глубокий семантический анализ (NLP) TF-IDF, Word2Vec.
